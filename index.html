Sincronização entre Dispositivos opcional

Por padrão, tudo no Life Jiu fica só no seu navegador. Se quiser acessar seus dados em outro celular/computador, conecte um banco gratuito do Supabase (supabase.com) — é a única forma de ter sincronização real sem depender de um servidor que a gente mesmo hospede. Você cria sua própria conta, então seus dados continuam sob seu controle, não nosso.

Como configurar (uma vez só): crie um projeto grátis em supabase.com → vá em SQL Editor e execute: create table lifejiu_sync (id text primary key, payload jsonb, updated_at timestamptz default now()); alter table lifejiu_sync enable row level security; create policy "allow all" on lifejiu_sync for all using (true); Depois cole a URL do projeto e a chave "anon public" abaixo (em Project Settings → API).
